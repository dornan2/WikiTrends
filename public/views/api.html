WikiTrends

API

Welcome to the WikiTrends API

Instructions on how to use the API are the following:

URL Description
/api/daily100 Returns an array of the top 100 articles viewed that day
/api/daily100/:position Returns the nth most popular article that day
/api/monthly100 Returns an array of the top 100 articles viewed that month
/api/monthly100/:position Returns the nth most popular article that month
/api/yearly100 Returns an array of the top 100 articles viewed that year
/api/yearly100/:position Returns the nth most popular article that year
/api/trending100 Returns an array of the top 100 trending articles that year
/api/trending100/:position Returns the nth top trending article that year
/api/articles/:articleName Returns all information on that individual article
/api/forecast Returns the forecasted top viewed articles in 2 weeks time
