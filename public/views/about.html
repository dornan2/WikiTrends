WikiTrends

About

Find out what's trending on Wikipedia. This application was developed by Adrian Dornan as part of his final year project in Computer Applications in DCU.

The user can view what is currently trending, what is popular and future forecasts. Graphs are used to visualy display the data to the user.

The application has a Python back end that parses and stores page view statistics of Wikipedia articles. A time series of all article views are stored in a MongoDB database and presented to users using a MEAN (Mongo, Express, Angular and Node) stack.
